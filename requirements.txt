# Requirements for UR5 Robotic Sorting System

## Robot Controller (Server)
robodk>=5.6.0

## AI Vision Client (Raspberry Pi)
ultralytics>=8.0.0
opencv-python>=4.8.0
numpy>=1.24.0
Pillow>=10.0.0

## Installation Instructions

### For Robot Controller:
```bash
pip install robodk
```

### For AI Client (Raspberry Pi):
```bash
cd Client_Ai_detector
./setup.sh
# Or manually:
pip install -r requirement.txt
```
